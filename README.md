# Python-para-Pesquisa-Academica
Diferentes versões de uma aplicação de web scrap em python. Essa aplicação foi, e esta sendo, construída com a colaboração de diversos estudantes que escolheram engajar com esse projeto de forma voluntaria. Todos os códigos aqui postados são abertos para qualquer interessados. Gratidão aos envolvidos permeia cada linha de código aqui presente.
